# SHIFT APP

Тестовое iOS-приложение для регистрации пользователя и отображения списка товаров из внешнего API.

## Требования

- iOS 15.0+
- Xcode 14.0+
- Swift 5.5+

## Архитектура

Приложение построено на архитектурном паттерне MVVM (Model-View-ViewModel).

- **Models** - структуры данных (User, Product)
- **Views** - SwiftUI экраны (RegistrationView, MainView, ProductCell)
- **ViewModels** - бизнес-логика (RegistrationViewModel, MainViewModel)
- **Services** - вспомогательные сервисы (NetworkService, ValidationService, StorageService)

## Функциональность

### Экран регистрации
- Поля ввода: имя, фамилия, дата рождения, пароль, подтверждение пароля
- Валидация полей в реальном времени с отображением ошибок под каждым полем
- Правила валидации:
  - Имя и фамилия: минимум 2 символа
  - Пароль: минимум 6 символов, как минимум одна цифра и одна заглавная буква
  - Возраст: 18 лет и старше
- Кнопка регистрации активна только при заполнении всех полей
- При успешной регистрации имя пользователя сохраняется в UserDefaults

### Главный экран
- Динамический список товаров, загружаемых с сервера
- API: https://fakestoreapi.com/products
- Отображается название товара и цена
- Кнопка "Приветствие" показывает модальное окно с именем пользователя
- Pull-to-refresh для обновления списка

### Сохранение сессии
- При успешной регистрации сохраняется статус авторизации
- При повторном запуске приложение открывается на главном экране

## Цветовая схема

- Основной цвет: #0E6973 (бирюзовый)
- Акцентный цвет: #FFD500 (жёлтый)


## Установка и запуск

1. Клонировать репозиторий
2. Открыть `SHIFT APP.xcodeproj` в Xcode
3. Выбрать целевое устройство (симулятор или реальное устройство)
4. Нажать `Cmd + R` для сборки и запуска

## Зависимости

Проект не использует сторонних зависимостей. Реализован на нативных фреймворках Apple: SwiftUI, Combine, Foundation.

## Тестирование

Unit-тесты покрывают логику валидации данных и работу сервисов.

Запуск тестов: `Cmd + U`
